# 东南亚知识产权案件参考

> 本文件供 legal-researcher 和相关 agent 在处理东南亚知识产权案件时按需加载。

## 核心法律框架

| 国家 | 主要立法 |
|------|---------|
| 新加坡 | Copyright Act 2021、Trade Marks Act 1998、Patents Act 1994、Registered Designs Act |
| 马来西亚 | Copyright Act 1987、Trade Marks Act 2019、Patents Act 1983 |
| 印尼 | UU No.28/2014 on Copyright、UU No.20/2016 on Marks and Geographical Indications、UU No.13/2016 on Patents |
| 泰国 | Copyright Act B.E. 2537、Trademark Act B.E. 2534、Patent Act B.E. 2522 |
| 越南 | Law on Intellectual Property 2005（2019/2022 改正） |
| 菲律宾 | Intellectual Property Code (RA 8293) |

## ASEAN 共通框架

- ASEAN Patent Examination Co-operation (ASPEC) — 东盟专利审查协作
- ASEAN Framework Agreement on IP Cooperation
- Madrid Protocol（多国加入 — 商标国际注册，except Myanmar 未加入）
- RCEP（Regional Comprehensive Economic Partnership — IP 章含 IP 执行标准）

## 常见争议焦点

### 商标权
- 商标恶意注册（bad faith filing — 中国企业在东南亚的 major risk）
- 不使用取消（通常 3-5 年）
- 周知商标保护（各国认定基准差异大）
- 近似判断在各国的标准不同

### 专利权
- 多数国家无实质审查（Indonesia "simple patent" — 仅形式审查 / 泰国 — 专利审查积压严重）
- 强制许可（compulsory license — 印尼/泰国过去曾对药品签发强制许可）
- 专利链接制度（菲律宾/新加坡/马来西亚等 linkage）
- 多数国家属 PCT 成员国

### 著作权
- 软件著作权
- 网络著作权侵害（Singapore 为较先进、越南/印尼 执行较弱）
- 邻接权（performing rights）

### 商业秘密
- 多数国家有一定保护但执行不一
- 新加坡、马来西亚（普通法 confidence）+ 有专门法
- 印尼（UU No.30/2000 on Trade Secrets）
- 越南（IP法含営業秘密の定義）

## 举证要点

| 争议类型 | 关键证据 |
|---------|---------|
| 商标恶意注册 | 使用证据、用户认知度调查 |
| 商标侵权 | 注册证、使用证据 |
| 专利侵权 | 专利授权书、claims chart |
| 商业秘密 | NDA、秘密管理规程证明 |

## 程序特点

- 新加坡 → 知识产权专门法院（IPOS + High Court）
- 马来西亚 → IP Court（吉隆坡高裁专门court）
- 泰国 → Central Intellectual Property and International Trade Court（知财通商专门裁判所 1997 年设立）
- 越南 → 行政権限（大部分 IPR enforcement 是行政侵害处理）+ 民事裁判 / 刑事
- 区域内跨境知识产权执行需考虑各国语言、文化、司法独立性

## 诉讼时效

| 国家 | IP 侵权损害赔偿时效 |
|------|-------------------|
| 新加坡 | 6 年（一般） |
| 马来西亚 | 6 年 |
| 印尼 | 一般消除时效（daluwarsa）30 年（少見）；侵权：不明（判例导向） |
| 泰国 | 3 年（知道侵害和加害者） |
| 越南 | 3 年（自知道权利被侵害起） |
