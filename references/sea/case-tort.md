# 东南亚侵权责任案件参考

> 本文件供 legal-researcher 和相关 agent 在处理东南亚侵权案件时按需加载。

## 核心法律框架

| 国家 | 主要来源 |
|------|---------|
| 新加坡 | Common law tort（negligence, nuisance, defamation, etc.）+ Defamation Act |
| 马来西亚 | Common law tort + Defamation Act 1957 |
| 印尼 | KUHPerdata（民法典）第 1365 条（PMH — Perbuatan Melawan Hukum — 侵权行为） |
| 泰国 | Civil and Commercial Code §420 以下的"不法行为"（ละเมิด） |
| 越南 | Civil Code 2015, Part IV (Obligations, art. 584 ff.) + Law on Protection of Consumer Rights |
| 菲律宾 | Civil Code（Title on Obligations and Contracts + Torts）+ Common law 影响 |

## 侵权构成要件

### 普通法系国家（新加坡/马来西亚）
- **Negligence** 四要件：Duty of Care + Breach of Duty + Causation + Damages
- 采用 Modified Caparo test / Singapore：Spandeck test
- 纯经济损失（pure economic loss）原则上不予赔偿

### 大陆法系国家
- **印尼（PMH — 民法典 1365 条）** — 类似荷兰 onrechtmatige daad
  - 广义"不法行为（melawan hukum）"包含成文法违反 + 违反不成文注意义务 + 违反社会适当性
- **泰国（CCC §420）** — 故意/过失侵害他人生命、身体、健康、自由、财产或权利
  - 最高法院判例扩展不法性范围
- **越南（Civil Code art. 584）** — 不法侵害生命、健康、名誉、财产、其他合法权益

## 损害赔偿

| 损害类型 | 说明 |
|---------|------|
| 实际损害 | 医疗费、财产损害等 |
| 逸失利益 | 丧失未来收入 |
| 精神损害 | 各国认定差异大 — 越南（Danh dự, nhân phẩm, uy tín）、泰国（精神损害在人身伤害案中认定）、新加坡（nervous shock 严格限制） |
| 惩罚性赔偿 | 基本不承认（普通法系在极少数案中可能） |

## 特殊制度

- **印尼** — Class action（集团诉讼 — 2002年最高法院规则）用于消费者/环境案
- **泰国** — 产品責任法：Product Liability Act B.E. 2551（2008年）— 类似 EU 严格责任
- **越南** — 产品责任在消费者保护法下

## 证明标准

- 普通法系：**balance of probabilities** (>50%)
- 大陆法系：**高度盖然性**程度
- 举证责任：主张者举证。过失推定（res ipsa loquitur）在普通法系适用

## 诉讼时效

| 国家 | 侵权时效 |
|------|---------|
| 新加坡 | 6 年（personal injury negligence — 3 年） |
| 马来西亚 | 6 年（一般）/ 3 年（人身伤害） |
| 印尼 | 民法典无专门侵权时效 → 一般消灭时效（daluwarsa）30 年，但判例适用 5 年 |
| 泰国 | 1 年（知道侵害和加害者）/ 10 年（行为时） — CCC §448 |
| 越南 | 3 年（知道权利受侵害起） |
