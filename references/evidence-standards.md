# 证据三性审查标准

> 本文件供 red-team-lawyer 在质证阶段按需加载。

## 真实性审查清单

| 检查项 | 攻击方向 | 适用证据类型 |
|--------|---------|-------------|
| 原件缺失 | 仅提供复印件/打印件，无法核对原件 | 书证、电子数据 |
| 来源不明 | 无法追溯证据形成过程 | 电子数据、录音录像 |
| 时间不明 | 无法确定形成时间或不同证据时间矛盾 | 全类型 |
| 制作主体不明 | 无法确认制作者身份 | 书证、电子数据 |
| 可篡改性 | 电子数据是否可被修改、有无哈希校验 | 电子数据 |
| 单方形成 | 对方不在场、不知情的情况下形成 | 书证、录音录像 |
| 缺少原始载体 | 电子数据未提供原始设备/存储介质 | 电子数据 |
| 鉴定资质 | 鉴定机构/人员是否具备相应资质 | 鉴定意见 |
| 证据完整性 | 录音录像是否经过剪辑、是否完整 | 录音录像 |

## 合法性审查清单

| 检查项 | 攻击方向 |
|--------|---------|
| 取证手段 | 偷拍偷录是否超出合法范围、是否严重侵害他人合法权益 |
| 程序合规 | 取证程序是否违反法律禁止性规定 |
| 隐私侵权 | 是否侵犯他人隐私权、通信自由 |
| 未经授权 | 是否未经许可获取他人信息或财物 |
| 证据形式 | 证据形式是否符合法定要求（如书面合同需原件） |

## 关联性审查清单

| 检查项 | 攻击方向 |
|--------|---------|
| 证明对象 | 证据证明的事实是否为本案争议事实 |
| 关联链条 | 从证据到待证事实之间是否存在断裂 |
| 推理跨度 | 从证据到结论需要多少假设性推论（推论越多证明力越弱） |
| 唯一性 | 证据是否可以得出其他合理解释（替代解释） |

## 证明力强弱参考

| 证明力等级 | 证据特征 |
|-----------|---------|
| 强 | 原件、第三方制作/认证、形成于争议发生前、有其他证据印证 |
| 中 | 复印件但有其他证据佐证、当事人自认部分事实、形成时间存疑 |
| 弱 | 孤证、单方制作、形成于争议发生后、无其他证据印证、可篡改 |
