# 合同纠纷案件参考

> 本文件供 legal-researcher 和相关 agent 在处理合同纠纷案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 《民法典》合同编（第三编） | 合同订立、效力、履行、变更、解除、违约责任 |
| 《民法典》总则编 | 民事法律行为效力、代理、诉讼时效 |
| 《电子商务法》 | 电子合同、格式条款 |

## 合同效力审查要点

| 效力类型 | 认定标准 |
|---------|---------|
| 无效 | 违反法律行政法规强制性规定、违背公序良俗、恶意串通损害他人利益 |
| 可撤销 | 重大误解、欺诈、胁迫、显失公平 |
| 效力待定 | 限制民事行为能力人订立、无权代理、无权处分 |

## 常见争议焦点

- 合同解除条件是否成就（法定解除 vs 约定解除）
- 违约金过高调整（以实际损失为基础，兼顾合同履行情况）
- 定金罚则适用
- 同时履行抗辩权 / 先履行抗辩权 / 不安抗辩权
- 情势变更 vs 商业风险
- 债权转让通知效力
- 格式条款效力（提示说明义务）

## 合同审查重点（供 evidence-mapper 合同审查模式使用）

- 条款无效风险：是否违反强制性规定
- 条款歧义风险：多种解释可能
- 举证风险：约定不明导致举证困难
- 管辖风险：管辖约定是否有效
- 付款风险：付款条件/期限是否明确
- 违约风险：违约金是否过高/过低
- 履约风险：履行标准是否可量化
- 撤销风险：是否存在可撤销事由
