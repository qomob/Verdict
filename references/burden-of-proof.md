# 举证责任分配规则

> 本文件供 judge 在评估举证责任时按需加载。

## 通用原则

### "谁主张，谁举证"

《民事诉讼法》第六十四条：当事人对自己提出的主张，有责任提供证据。

### 举证责任倒置

| 案件类型 | 倒置规则 | 法律依据 |
|---------|---------|---------|
| 医疗损害责任 | 医疗机构就诊疗行为无过错、与损害无因果关系举证 | 《民法典》第1222条 |
| 环境污染 | 污染者就行为与损害无因果关系举证 | 《民法典》第1230条 |
| 高度危险作业 | 经营者就受害人故意举证 | 《民法典》第1238-1244条 |
| 建筑物倒塌 | 建设单位、施工单位就不存在质量问题举证 | 《民法典》第1252条 |
| 饲养动物损害 | 动物饲养人就受害人故意或重大过失举证 | 《民法典》第1245-1246条 |

### 劳动争议特殊规则

| 争议类型 | 举证责任方 | 法律依据 |
|---------|-----------|---------|
| 工资支付记录 | 用人单位（保存2年） | 《工资支付暂行规定》第6条 |
| 考勤记录 | 用人单位 | 《劳动争议调解仲裁法》第6条 |
| 解除劳动合同原因 | 用人单位 | 《最高人民法院劳动争议司法解释（一）》第44条 |
| 加班事实 | 劳动者主张→用人单位提供考勤 | 《劳动争议调解仲裁法》第6条 |
| 规章制度公示 | 用人单位 | 《劳动合同法》第4条 |

## 证明标准

| 标准 | 适用场景 | 要求 |
|------|---------|------|
| 高度盖然性 | 一般民事案件 | 证据使待证事实的存在具有高度可能性（>75%） |
| 较高盖然性 | 部分民事案件 | 证据使待证事实的存在具有较大可能性（>60%） |
| 盖然性 | 程序性事项 | 证据使待证事实的存在具有可能性（>50%） |

## 举证不能的后果

- 负有举证责任的一方未能提供充分证据 → 承担不利后果
- 证据不足以证明待证事实 → 该事实不予认定
- 关键证据灭失且无法补救 → 可能导致败诉

## 证据采信优先级

| 优先级 | 证据类型 |
|--------|---------|
| 1 | 公证文书、生效裁判文书、仲裁裁决书 |
| 2 | 原件书证、经公证的电子数据 |
| 3 | 经质证的证人证言、当事人陈述 |
| 4 | 复印件（与其他证据印证） |
| 5 | 孤证（无其他证据印证） |
