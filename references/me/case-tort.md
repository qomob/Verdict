# 中东侵权责任案件参考

> 本文件供 legal-researcher 和相关 agent 在处理中东侵权案件时按需加载。

## 核心法律框架

| 法域 | 主要来源 |
|------|---------|
| UAE | Civil Transactions Law — 侵权行为（taqsir / al-fi'l al-darr） |
| 沙特 | Shari'ah 损害赔偿原则 + 部分成文商业法 |
| 卡塔尔 | Civil Code — 侵权条款 |
| DIFC | DIFC Law of Obligations（基于 English law of tort） |
| 埃及 | Civil Code（Egyptian Civil Code art. 163 ff. — 不法行为） |

## 侵权构成要件（大陆法系 / 伊斯兰法系共通的 3 要素）

1. **Al-khata'（过错 / 不法行为）** — UAE 民事交易法：广义过错 = 违反法律义务（与法国 model 相近）
2. **Al-darar（损害）** — 财产损害 + 身体损害 + 精神损害（al-darar al-adabi）
3. **Al-'alaqa al-sababiyya（因果关系）** — 事实因果 + 相当因果

**Shari'ah 损害赔偿计算的特殊性：** diya（血金 / 身体损害的对价 — 基于 Islamic jurisprudence 的传统赔偿额）

## 特殊责任

| 类型 | 说明 |
|------|------|
| 使用者责任 | 雇主/业主对雇员的侵权行为负责（UAE 民交法 close to French model） |
| 建築物責任 | 建筑物倒塌/瑕疵所致人损 — 所有者责任 |
| 物权侵害（trespass）| DIFC / UAE |
| 产品責任 | 中东多数国家无专门产品责任法 — 基于一般侵权 |

## 损害赔偿

| 类型 | 说明 |
|------|------|
| 财产损害 | 实际损害 + 可得利益丧失 |
| 身体损害 | 医疗费 + 逸失利益 + diya |
| 精神损害 | 在伊斯兰法/阿拉伯法中承认（al-darar al-adabi）— UAE 民交法 art. 293 |
| 惩罚性赔偿 | 原则上不承认 |

## 证明标准

- UAE / 卡塔尔 / 沙特 → **法官内心确信 (al-iqtina' al-shakhsi)**
- DIFC → **Balance of probabilities**

## 诉讼时效

| 法域 | 时效 |
|------|------|
| UAE | 3 年（知道损害和加害者之日） |
| 沙特 | 不确定（Shari'ah 无消灭时效概念，但近年实践应用一定限制） |
| 卡塔尔 | 3 年 |
| DIFC | 3 年（人身伤害 — DIFC Law of Obligations） |
| 埃及 | 3 年 |
