# 股权纠纷案件参考

> 本文件供 legal-researcher 和相关 agent 在处理股权纠纷案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 《公司法》（2023修订） | 股权确认、股权转让、股东权利义务、公司治理 |
| 《民法典》合同编 | 股权转让合同效力 |
| 《证券法》 | 上市公司股权相关 |

## 司法解释

- 《最高人民法院关于适用〈中华人民共和国公司法〉若干问题的规定（一）至（五）》
- 《最高人民法院关于审理外商投资企业纠纷案件若干问题的规定》

## 常见争议焦点

- 股权确认（隐名股东 vs 显名股东）
- 股权转让合同效力（优先购买权、转让限制）
- 股东出资纠纷（虚假出资、抽逃出资）
- 股东知情权诉讼
- 公司决议效力（股东会/董事会决议无效/可撤销）
- 股东利润分配请求权
- 股东代表诉讼（损害公司利益）

## 举证要点

| 争议类型 | 关键证据 |
|---------|---------|
| 隐名股东确认 | 代持协议、实际出资凭证、参与经营管理证据 |
| 股权转让效力 | 转让协议、通知其他股东记录、其他股东放弃优先购买权声明 |
| 出资不实 | 验资报告、银行流水、资产评估报告 |
| 决议效力 | 会议通知、会议记录、表决程序、决议内容 |
| 知情权行使 | 股东资格证明、已履行前置程序（书面请求） |
| 抽逃出资 | 转账记录、会计凭证、审计报告 |

## 特别注意

- 2023年新《公司法》2024年7月1日施行，注意新旧法衔接
- 股东纠纷往往涉及公司内部治理文件（章程、协议），需结合章程条款分析
- 股权确认案件需区分"内部纠纷"（股东之间）与"外部纠纷"（股东与第三人）
