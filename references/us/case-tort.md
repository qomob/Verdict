# 美国侵权责任案件参考

> 本文件供 legal-researcher 和相关 agent 在处理美国侵权案件时按需加载。

## 核心法律框架

| 法律渊源 | 适用场景 |
|---------|---------|
| Restatement (Third) of Torts | 侵权一般原则（各州普遍引用） |
| State Tort Laws | 各州判例法和成文法（多数侵权法是州法，非联邦法） |
| Federal Tort Claims Act (FTCA) | 联邦政府侵权责任 |
| 42 U.S.C. § 1983 | 州政府人员侵犯联邦宪法权利 |

## 侵权类型

### 故意侵权 (Intentional Torts)
- Battery — 故意有害/冒犯接触
- Assault — 合理恐惧即将发生的接触
- False Imprisonment — 非法拘禁
- Intentional Infliction of Emotional Distress (IIED) — 极端且肆意的行为造成严重精神损害
- Trespass to Land / Chattels — 侵犯土地/动产
- Defamation — 诽谤（libel 书面 / slander 口头）；公共人物需证明 actual malice（New York Times v. Sullivan）

### 过失侵权 (Negligence)
四大要件：
1. **Duty（注意义务）** — 合理人标准 / 特殊情况（premises liability based on entrant status: invitee / licensee / trespasser）
2. **Breach（违反义务）** — 行为低于合理标准 / negligence per se（违反成文法即推定过失）/ Res ipsa loquitur
3. **Causation（因果关系）** — 事实因果（but-for test / substantial factor）+ 法律因果（proximate cause / foreseeability）
4. **Damages（损害）** — 实际损害 / 纯经济损失一般不可诉（economic loss rule）

### 严格责任 (Strict Liability)
- 异常危险活动（abnormally dangerous activities）
- 产品责任（Restatement Third: manufacturing defect / design defect / failure to warn）
- 动物致害（野生动物 + 知悉危险倾向的家养动物）

## 损害赔偿

| 类型 | 内容 |
|------|------|
| Compensatory damages | 经济损失（医疗费/误工费）+ 非经济损失（pain and suffering） |
| Punitive damages | 惩罚性赔偿（需证明恶意/重大过失/鲁莽，多数州有上限或 constitutional limits — BMW v. Gore 指引） |
| Wrongful death | 过失致死诉讼由家属提起 |
| Joint and several liability | 共同侵权连带责任 — 多数州已修改为比较过错下的按份责任 |

## 抗辩事由

- 比较过错（comparative fault — 多数州：纯比较 < pure > vs 修正比较 < modified 50%/51% bar >）
- 自担风险（assumption of risk — express / implied primary / implied secondary）
- 诉讼时效（statute of limitations — 各州不同，通常是受伤之日起 1-3 年，discovery rule 延长）

## 证明标准

民事侵权适用 **preponderance of the evidence**（证据优势标准，>50%）。clear and convincing evidence 适用于 punitive damages 和部分 fraud 案件。
