# 美国知识产权案件参考

> 本文件供 legal-researcher 和相关 agent 在处理美国知识产权案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| Copyright Act (17 U.S.C.) | 著作权（原创作品固定于有形载体） |
| Lanham Act (15 U.S.C.) | 商标权、商业外观、虚假广告 |
| Patent Act (35 U.S.C.) | 发明专利、实用新型（无中国式实用新型，但有 design patent） |
| Defend Trade Secrets Act (DTSA) + State UTSA | 商业秘密 |
| Digital Millennium Copyright Act (DMCA) | 网络著作权、安全港、反规避 |

## 常见争议焦点

### 著作权
- 固定性（fixation）和原创性（originality）
- 侵权认定：实质性相似（substantial similarity）+ 接触（access）
- 合理使用四要素（17 U.S.C. § 107）：使用目的、作品性质、使用量、市场影响
- 首次销售原则（first sale doctrine）
- 雇佣作品（work made for hire）
- DMCA Safe Harbor（§ 512）：通知-删除规则

### 商标权
- 显著性（distinctiveness）：generic / descriptive / suggestive / arbitrary / fanciful
- 侵权认定：likelihood of confusion（混淆可能性的多因素测试 — Polaroid/Sleekcraft factors）
- 淡化（dilution）：blurring / tarnishment — 仅适用于著名商标
- 商业外观（trade dress）
- 合理使用（nominative fair use / descriptive fair use）

### 专利权
- 可专利性：novelty / non-obviousness（非显而易见性）/ utility
- 权利要求解释：Markman hearing（法官解释权利要求范围）
- 侵权理论：literal infringement / doctrine of equivalents（等同原则）
- 故意侵权（willful infringement）→ enhanced damages up to 3x
- 专利无效抗辩：Alice test（抽象概念不属于可专利客体）

### 商业秘密
- DTSA 要素：信息有独立经济价值 + 采取了合理保密措施
- 不当获取（misappropriation）：不正当手段获取 / 违反保密义务
- 救济：injunction / damages / 单方扣押令（ex parte seizure in extraordinary circumstances）

## 举证要点

| 争议类型 | 关键证据 |
|---------|---------|
| 著作权侵权 | 著作权登记证书（注册是起诉前提）、接触证据、实质性相似比对 |
| 商标侵权 | 商标注册证、使用证据、知名度/第二含义证据、消费者混淆调查（survey evidence） |
| 专利侵权 | 专利授权文件、权利要求表（claims chart）、被控侵权产品技术特征分析 |
| 商业秘密 | 保密协议（NDA）、访问日志、保护措施文档、被告接触证据 |

## 赔偿计算

- 著作权：实际损害 + 侵权人获利 / 法定赔偿 $750-$30,000（故意最高 $150,000）
- 商标：实际损害 + 侵权人获利 / 法定赔偿 $1,000-$200,000（假冒故意最高 $2,000,000）
- 专利：合理许可费（reasonable royalty）+ 故意侵权三倍
- 商业秘密：实际损失 + 不当得利 / 合理许可费 + 故意窃取两倍惩罚性赔偿

## 诉讼时效

| 类型 | 时效 |
|------|------|
| 著作权 | 3 年（发现之日起） |
| 商标（Lanham Act） | 无成文法时效，按 laches（懈怠）原则参照州法 |
| 专利 | 6 年（损害赔偿追溯） |
| 商业秘密（DTSA） | 3 年（发现之日起） |
