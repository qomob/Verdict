# 美国劳动争议案件参考

> 本文件供 legal-researcher 和相关 agent 在处理美国劳动/雇佣纠纷案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| Fair Labor Standards Act (FLSA) | 最低工资、加班费（overtime）、童工 |
| Title VII of Civil Rights Act of 1964 | 就业歧视（种族、肤色、宗教、性别、国籍） |
| Age Discrimination in Employment Act (ADEA) | 年龄歧视（40 岁以上） |
| Americans with Disabilities Act (ADA) | 残疾歧视、合理便利（reasonable accommodation） |
| Family and Medical Leave Act (FMLA) | 家庭和医疗假期（最多 12 周无薪） |
| National Labor Relations Act (NLRA) | 工会权利、集体谈判、不当劳动行为 |
| Occupational Safety and Health Act (OSHA) | 工作场所安全 |
| Worker Adjustment and Retraining Notification Act (WARN) | 大规模裁员提前通知（60 天） |
| Equal Pay Act | 同工同酬（性别） |

## 雇佣关系原则

- **At-Will Employment（任意雇佣）：** 多数州默认雇佣关系可被任意一方随时终止，无需理由。例外：合同约定、公共政策例外、默示合同例外、诚信与公平交易例外。
- **Wrongful Termination（不当解雇）：** 违反歧视法律、违反公共政策（whistleblower retaliation）、违反合同约定。

## 举证责任

| 争议类型 | 举证结构 |
|---------|---------|
| 歧视（disparate treatment） | McDonnell Douglas 三阶段：原告建立 prima facie case → 雇主给出合法非歧视理由 → 原告证明理由是 pretext（借口） |
| 工资工时（FLSA） | 雇员证明未获最低工资/加班费 → 雇主承担记录保存义务（FLSA recordkeeping） |
| 骚扰（harassment） | 雇员证明 harassment 严重或普遍（severe or pervasive）+ 雇主知悉但未采取纠正措施 |
| 报复（retaliation） | 雇员证明 protected activity → adverse action → causal connection |

## 常见争议焦点

- 加班费豁免（exempt vs non-exempt）：executive / administrative / professional / computer employee / outside sales
- 独立承包商 vs 雇员（ABC test / economic realities test）
- 最低工资和 tip credit
- 歧视案的 disparate impact vs disparate treatment
- Hostile work environment（敌意工作环境）
- Constructive discharge（推定解雇）
- 竞业限制（non-compete）— FTC 2024 规则禁止大多数 non-compete，各州法律不同
- 性骚扰与 Quid Pro Quo harassment

## 救济方式

| 救济 | 内容 |
|------|------|
| Back pay | 欠薪 |
| Front pay | 未来薪资损失（代替复职） |
| Compensatory damages | 补偿性赔偿（精神损害等） |
| Punitive damages | 惩罚性赔偿（歧视案中 capped at $50k-$300k，取决于雇主规模） |
| Reinstatement | 复职 |
| Attorney's fees | 律师费（胜诉方在多数联邦反歧视法中可获律师费） |
| Liquidated damages | 双倍赔偿（FLSA 欠薪默认 doubled，除非雇主证明善意） |

## 行政前置程序

| 争议类型 | 前置程序 |
|---------|---------|
| 就业歧视 | EEOC Charge（先向 EEOC 投诉，取得 Right-to-Sue Letter 后方可起诉）— 180-300 天时效 |
| FLSA 工资争议 | 可直接起诉，无需行政前置 |
| NLRB 管辖 | 工会/集体谈判相关需先向 NLRB 提起 unfair labor practice charge |
