# 劳动争议案件参考

> 本文件供 legal-researcher 和相关 agent 在处理劳动争议案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 《劳动合同法》 | 劳动合同订立、履行、变更、解除、终止 |
| 《劳动法》 | 劳动关系基础规范 |
| 《劳动争议调解仲裁法》 | 仲裁程序、举证规则、时效 |
| 《社会保险法》 | 社保争议 |
| 《工资支付暂行规定》 | 工资支付、扣减 |

## 举证责任特殊规则

| 争议类型 | 举证责任方 |
|---------|-----------|
| 工资支付记录 | 用人单位（保存2年） |
| 考勤记录 | 用人单位 |
| 解除劳动合同原因 | 用人单位 |
| 加班事实 | 劳动者主张→用人单位提供考勤 |
| 规章制度公示 | 用人单位 |
| 工伤认定 | 用人单位在30日内申请 |

## 常见争议焦点

- 违法解除劳动合同赔偿金（2N）vs 经济补偿金（N）
- 未签书面合同二倍工资
- 加班费计算基数
- 竞业限制补偿与违约
- 年休假折算
- 社保补缴（属于行政范畴，部分法院不受理）
- 工伤待遇（先认定工伤，再主张待遇）

## 仲裁前置

劳动争议实行"仲裁前置"原则，必须先经劳动仲裁，对仲裁裁决不服才能起诉。部分案件（追索劳动报酬不超过当地月最低工资标准12个月金额的争议）适用终局裁决。
