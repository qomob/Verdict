# 韩国劳动案件参考

> 本文件供 legal-researcher 和相关 agent 在处理韩国劳动纠纷案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 근로기준법（劳动基准法） | 最低劳动条件（工资、工时、解雇） |
| 근로자퇴직급여 보장법 | 退休金 |
| 노동조합 및 노동관계조정법（労組法） | 工会、集体谈判、不当劳动行为 |
| 남녀고용평등법 | 男女就业平等、性骚扰禁止 |
| 산업안전보건법 | 产业安全保健 |
| 최저임금법 | 最低工资 |
| 기간제 및 단시간근로자 보호 등에 관한 법률（기간제법） | 非正规职保护（固定期限、短时间劳动者） |
| 파견근로자보호 등에 관한 법률（파견법） | 派遣劳动者保护 |
| 중대재해 처벌 등에 관한 법률（中灾法） | 重大灾害企业处罚法（2022年施行—严重工伤时对经营责任人的刑事处罚） |

## 解雇法理

韩国采用**解雇正当性审查**（근로기준법 제23조 제1항）：
- 解雇需有「정당한 이유」（正当理由）
- 普通解雇（통상해고）：客观合理理由 + 社会通念上相当性
- 整理解雇（경영상 해고）：紧急的经营必要性 + 努力避免解雇 + 合理的公正的选择标准 + 与工会/劳动者诚实协议（50 天前通知）
- 惩戒解雇（징계해고）：需比普通解雇更重的理由，且需正当程序（취업규칙等规定的程序）
- 解雇无效 → 劳动关系存续 + 中间工资全额 + 复职（大法院强烈倾向复职）

## 举证责任

| 争议类型 | 举证方 |
|---------|--------|
| 解雇正当理由 | 雇主（使用者） |
| 未支付工资 | 劳动者证明劳动事实 → 雇主承担支付事实 |
| 加班费 | 劳动者主张 → 雇主持有出勤记录义务 |
| 性骚扰 | 劳动者证明事实 → 雇主承担防止义务举证 |
| 工伤（産業災害） | 劳动者接近因果关系的证明 → 勤劳福祉公団认定 |

## 常见争议焦点

- 不正当解雇（부당해고）— 最重要的劳动诉讼类型
- 未支付工资 + 지연이자（迟延利息 — 年 20%）
- 退职金未支付
- 雇佣形态（정규직 / 비정규직）— 固定期限合同的 갱신기대권（更新期待权）
- 工作时间与加班费 — 週52時間上限制（근로기준법 제53조）
- 직장 내 괴롭힘（职场内欺凌 — 2019年 근로기준법改正禁止）
- 不当劳动行为（부당노동행위）— 工会关联

## 救济方式

| 救济 | 说明 |
|------|------|
| 解雇无效确认 + 工资请求 | 主要诉讼类型 |
| 부당해고 구제신청（劳动委员会） | 行政救济 — 60天时效（快速） |
| 未付工资 + 지연이자 | 年 20%（근로기준법 제37조） |
| 退职金 + 利息 | |
| 慰謝料（정신적 손해） | 精神损害赔偿 |

## 程序特点

- **劳动委员会前置（부당해고 구제신청）** — 60 天时效！过时效可直接向法院起诉（无前置限制）
- 地方法院劳动案件集中管辖
- 劳动委员会의 구제명령 → 中央劳动委员会再審 → 行政诉讼（法院）
- 근로감독관（劳动监督官 — 고용노동부）的行政监督和是正命令

## 诉讼时效

| 类型 | 时效 |
|------|------|
| 工资请求 | 3 年 |
| 退职金 | 3 年 |
| 解雇无效确认 | 无除斥期间（2021年 대법원 전원합의체 判例确认） |
| 损害赔偿（侵权） | 3 年 / 10 年 |
