# 韩国侵权责任案件参考

> 本文件供 legal-researcher 和相关 agent 在处理韩国侵权案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 민법 第750条〜766条 | 不法行为基本规则 |
| 제조물책임법 | 产品责任 — 缺陷制造者严格责任 |
| 자동차손해배상 보장법（自赔法） | 交通事故 — 运行者无过失责任（近乎绝对责任） |
| 국가배상법 | 国家赔偿 |
| 개인정보 보호법 | 个人信息侵害 |

## 不法行为成立要件（民法750条）

1. **고의 또는 과실（故意或过失）** — 客观注意义务违反
2. **위법성（违法性）** — 权利侵害或法益侵害
3. **손해의 발생（损害发生）** — 财产损害 + 精神损害（위자료）
4. **인과관계（因果关系）** — 事实因果（条件关系）+ 相当因果关系（赔偿范围限定）

## 特殊不法行为

| 类型 | 内容 | 依据 |
|------|------|------|
| 使用者责任（사용자책임） | 雇主对雇员执行职务所致损害负责 — 免責极难（选任/监督无过失都难成立） | 民法756条 |
| 工作物责任（공작물책임） | 构筑物瑕疵所致损害 — 占有者一次责任，所有者二次责任 | 民法758条 |
| 共同不法行为（공동불법행위） | 连带责任（부진정연대채무 — 不真正连带债务）| 民法760条 |
| 监督者责任（감독자책임） | 无责任能力人致害时监督义务人责任 | 民法755条 |
| 产品责任 | 제조물책임법（产品责任法）— 缺陷制造者责任。抗弁：개발위험의 항변 | PL法 |

## 损害赔偿

| 损害赔偿类型 | 内容 |
|-------------|------|
| 적극적 손해（积极损害） | 医疗费、护理费、器具费等 |
| 소극적 손해（消极损害） | 逸失利益（일실수입）— 以实际收入为基准计算 |
| 위자료（慰藉料） | 精神损害赔偿 |
| 과실상계（过失相抵） | 受害人与有过失时减少赔偿额 |
| 손익상계（损益相抵） | 抵扣已获利益 |

## 诉讼时效

| 类型 | 时效 |
|------|------|
| 不法行为损害赔偿 | 3 年（知道损害和加害者）/ 10 年（行为时起） |
| 产品责任 | 3 年（知道损害和加害者）/ 10 年（产品供给时起 — 累计受害时 10 年） |

## 证明标准

- 民事证明度：고도의 개연성（高度盖然性 — 70-80% 程度の心証）
- 医疗诉讼：患者方证明减轻（大法院判例的"临床上因果关系的推定"）
- 环境污染：역학적 인과관계（疫学因果关系）可成立（大法院 2002.10.11. 2000다52244）

## 主要判例趋势

- 대법원 2019. 2. 14. 선고 2017다281956 — 共同不法行为与 과실상계
- 대법원 2006. 3. 9. 선고 2005다10030 — 使用者責任的免責范围
