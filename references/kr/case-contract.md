# 韩国合同纠纷案件参考

> 本文件供 legal-researcher 和相关 agent 在处理韩国合同纠纷案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 민법（民法典）第3编 债权 | 合同通则、买卖、租赁、雇佣、承揽等 |
| 상법（商法） | 商行为 |
| 약관의 규제에 관한 법률（约款规制法） | 格式条款规制 |
| 전자문서 및 전자거래 기본법 | 电子交易、电子文书 |
| 할부거래에 관한 법률 | 分期付款交易 |
| 방문판매 등에 관한 법률 | 访问贩卖、电话劝诱贩卖 |

## 合同效力

| 效力类型 | 认定标准 |
|---------|---------|
| 无效 | 违反强行法规、反社会秩序（반사회질서 — 民法103条）、不公正法律行为（불공정한 법률행위 — 民法104条） |
| 可撤销 | 错误（착오 — 民法109条）、欺诈/胁迫（사기·강박 — 民法110条） |
| 无权代理 | 表达代理（표현대리）、无权代理（무권대리）的制度 |

## 常见争议焦点

- 债务不履行（채무불이행）— 履行请求 + 损害赔偿（民法390条）
- 合同解除：履行迟延的催告解除 + 不能履行/拒绝履行的无催告解除
- 同时履行抗辩权（동시이행의 항변권 — 民法536条）
- 瑕疵担保责任（하자담보책임）— 买卖（民法580条），承揽（民法667条）
- 违约金（위약금）— 推定为损害赔偿预定额，过高时法院可适当减额（民法398条）
- 定金（계약금）：解约定金（해약금 — 民法565条）— 交付方抛弃 / 收受方加倍返还可解除
- 事情变更原则（사정변경의 원칙）— 判例法理（大法院承认但从严适用）
- 诚信原则（신의성실의 원칙 — 民法2条1项）+ 禁止权利滥用（民法2条2项）

## 合同审查重点（供 evidence-mapper 合同审查模式使用）

- 无效率由：反社会秩序（如过度限制竞争、暴利行为）
- 歧义风险：约款情况下对消费者有利解释
- 举证风险：主張하는 자（主张方）举证
- 管辖风险：专属管辖约定是否有效
- 付款风险：同时履行抗弁权
- 违约风险：损害赔偿范围 — 通常损害（통상손해）+ 特别损害但预见可能者为限（특별손해 — 民法393条）
- 履约风险：不確実한 경우의 履行拒绝权
- 撤销风险：错误（但表意者有重大过失时不可主张 — 民法109条但書）

## 诉讼时效

| 类型 | 时效 |
|------|------|
| 一般债权 | 10 年（民法162条） |
| 商事债权 | 5 年（商法64条） |
| 合同价款（工资、餐饮费等） | 3 年（民法163条1号 — 短期消灭时效） |
| 建設瑕疵（承揽） | 5 年（民法671条，铁筋造等坚固构筑物 10 年） |
