# 韩国举证责任与证据规则

> 本文件供 judge 和 case-intake 在评估韩国法管辖案件时按需加载。

## 通用原则

### 举证责任（입증책임）

韩国民事诉讼法采用 **法律要件分类说**（법률요건분류설）：
- 主张权利发生的一方 → 举证权利发生的要件事实
- 主张权利妨碍/消灭/排除的一方 → 举证相应的要件事实

### 证明标准

| 标准 | 适用场景 | 说明 |
|------|---------|------|
| 고도의 개연성（高度盖然性） | 一般民事案件 | 法官形成确信，不需自然科学的精确性 |
| 소명（疎明） | 保全处分等程序性事项 | 一応証明（初步证明） |

### 证据法则

- **自由心证主义（자유심증주의，民事诉讼法第202条）** — 法官自由判断证据价值
- 违法收集证据（위법수집증거）— 原则上排除；但在民事诉讼中判别是否排除时考虑：违法程度 + 证据重要性 + 案件性质
- 处分文书（처분문서，如合同）— 原则上不容许反证推翻书面内容
- 当事人本人询问 — 补充性证据

## 举证责任转换（입증책임의 전환）

| 特殊类型 | 举证规则 | 法律依据 |
|---------|---------|---------|
| 医疗过失 | 患者证明医疗行为 + 损害 → 医疗方证明无过失（大法院判例形成举证责任缓和 / 转换理论） | 判例 |
| 产品责任 | 製造物責任法：原告证明缺陷 + 损害 + 因果关系 → 被告证明免责事由（開発危険の抗弁等） | 제조물책임법 |
| 环境污染 | 대법원 판례：疫学因果关系可代个别因果关系证明 | 判例 |
| 劳动争议 | 解雇理由由用人单位举证 | 근로기준법 제27조 |

## 证据收集手段

| 手段 | 说明 |
|------|------|
| 문서제출명령（民事诉讼法第343条） | 法院命令持有文书方提交（范围较日本广 — 引用文书 / 交付文书 / 利益共通文书等均含） |
| 증거보전（民事诉讼法第375条） | 诉前证据保全 |
| 사실조회제도（事实查询） | 律师通过法院向公共机构查询 |
| 증인신문 / 당사자신문 | 证人询问 / 当事人本人询问 |

## 诉讼时效（소멸시효）

| 类型 | 时效 | 依据 |
|------|------|------|
| 一般债权 | 10 年（权利可行使起） | 민법 제162조 |
| 商事债权 | 5 年 | 상법 제64조 |
| 短期债权（工资、餐费等） | 3 年 | 민법 제163조 |
| 不法行为损害赔偿 | 3 年 / 10 年（知道损害和加害者 → 3 年 / 行为时 → 10 年） | 민법 제766조 |
| 使用人（雇主）过失的求偿权 | 3 年 | 민법 제766조 |
| 人身损害赔偿 | 同上（不法行为）| 민법 제766조 |

**时效中断事由：** 催告（최고，6 个月效力）、扣押/假扣押/假处分、承认（승인）

## 程序特点

- 三审制：지방법원 → 고등법원 → 대법원
- 대법원为法律审（原则上）
- 每审级可提出新证据（无严格失效效）
- 诉额 2 亿韩元以下由지방법원单独部管辖 / 2 亿以上由合议부管辖
