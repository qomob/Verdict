# 韩国公司/股权纠纷案件参考

> 本文件供 legal-researcher 和相关 agent 在处理韩国公司/股权纠纷案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 상법 第3编 会社（2011年大幅改正） | 株式会社、有限会社等公司治理 |
| 자본시장과 금융투자업에 관한 법률（资本市场法） | 上市公司信息披露、不公平交易 |
| 외부감사법（外部监查法） | 外部审计 |
| 공정거래법（公平交易法） | 大企业集团規制、经济力集中 |

## 常见争议焦点

### 股东纠纷
- 股东代表诉讼（주주대표소송 — 商法403条）— 持股 1%（上市 6 个月）+ 30天前提诉请求
- 股东知情权（회계장부열람권 — 商法466条）— 持股 3% 以上 + 记载理由
- 新股发行差止（신주발행유지청구）— 商法424条
- 少数股东权：累积投票请求权 등
- 组织重组中的反対股东股票买入请求权（반대주주의 주식매수청구권）
- 이사의 회사에 대한 책임 추궁

### 公司治理
- 董事的善管注意义务（선관주의의무）/ 忠实义务（충실의무 — 商法382条3项、382条の2）
- 业务执行指示者等的責任（商法401条の2）
- 内部統制基準（내부통제기준）
- 監事 / 監査委員会（감사위원회）的监督机制

### 并购纠纷
- MBO 中董事的利益冲突与公平性
- 合併（합병）· 분할（分立）等組織再編的价格公正性
- 少数股东 squeeze-out 制度（2011年商法改正導入）

## 举证要点

| 争议类型 | 关键证据 |
|---------|---------|
| 代表诉讼 | 提訴請求書 + 公司不提起理由 |
| 帳簿閲覧 | 股东资格 + 请求理由 |
| 董事责任 | 董事会会议录、决策过程资料 |
| 信息披露违法 | 事业报告书、公示资料 |

## 特别注意事项

- 大企業集团（재벌）内部交易的公正性审查（공정거래법 + 商法自我交易規制）
- 非上市公司股权评估方法
- 株主総会決議取消의 소 제기기간（决议日起 2 个月 — 商法376条）
- 新株発行无效之诉（决议日起 3 个月 — 商法429条）
- 集中投票制（집중투표제）可选择排除（定款可排除）

## 诉讼时效

| 类型 | 时效 |
|------|------|
| 董事对公司的责任（代表诉讼追究） | 10 年（商法 — 行为时） |
| 決議取消之诉 | 决议日起 2 个月 |
| 新株発行无效之诉 | 发行日起 3 个月 |
| 不法行为损害赔偿 | 3 年 / 10 年 |
