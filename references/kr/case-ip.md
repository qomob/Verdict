# 韩国知识产权案件参考

> 本文件供 legal-researcher 和相关 agent 在处理韩国知识产权案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 저작권법（著作权法） | 著作权 |
| 특허법（特许法） | 发明专利 |
| 실용신안법（实用新型法） | 实用新型 |
| 상표법（商标法） | 商标 |
| 디자인보호법（设计保护法） | 外观设计 |
| 부정경쟁방지 및 영업비밀보호에 관한 법률（不正当竞争防止法） | 商业秘密、商品表示混同、形态模仿 |

## 常见争议焦点

### 著作权
- 创作性认定（창작성）
- 侵害认定：의거관계（依拠关系）+ 실질적 유사성（实质性相似）
- 职务著作（업무상저작물）的归属
- 著作权侵害 → 民事（差止+損害赔償）+ 刑事（친고죄 → 2019年改正 非亲告罪化部分）

### 商标权
- 商标类似判断：外観·称呼·観念的综合
- 不使用取消審判（不使用者对注册商标的挑战 — 3 年以上未在韩国使用）
- 周知/著名商标的保护强化（부정경쟁방지법 제2조1호(차)목 等）
- 使用権者（통상사용권/전용사용권）

### 专利权
- 進歩性（진보성）判断
- 均等論（균등론 — 대법원 2000후3272等）
- 无效審判の对抗（특허무효심판 vs 侵害诉讼中的无效抗辩—2017年 대법원 판례 明确）
- 损害赔偿额算定：特許法 제128条 — 侵权者利益推定为损害额 / 实施料相当额 / 加重赔偿（최대 3배，2018年改正）
- **Punitive damages** → 2018年特許法改正：故意侵害最多 3 倍赔偿（징벌적 손해배상）

### 商业秘密（영업비밀）
- 3 要件：비밀관리성（秘密管理性）+ 유용성（有用性）+ 비공지성（非公知性）
- 2019年不正当竞争防止法改正：惩罚性赔偿最高 3 倍（故意侵害 영업비밀）
- 刑事处罚：최고 징역 15年（18억 원 이상 이득 시）

## 举证要点

| 争议类型 | 关键证据 |
|---------|---------|
| 著作权侵权 | 创作底稿、依拠证据、实质性相似证据 |
| 商标侵权 | 商标注册证、使用证据、周知性证据 |
| 专利侵权 | 特許公報、クレーム对照表、无效化资料 |
| 商业秘密 | 秘密管理規程、접근기록、NDA |

## 诉讼时效

| 类型 | 时效 |
|------|------|
| IP 侵权损害赔偿 | 3 年（知道损害和加害者）/ 10 年（行为时） |
| 不当利得 | 5 年 / 10 年 |

## 程序特点

- 特许法院（특허법원）— 专利/商标无效审判撤销诉讼的专属二审法院（대전 所在）
- 专利侵权一审：首尔中央地方法院等 5 个地方法院集中管辖（2016年改正 → 全国 6 个地方法院合议부）
- 特许厅（특허청 / KIPO）の 심판절차（审判程序）：无效审判 / 权利范围确认审判
- 特许审判院（특허심판원）内部审理
