# 欧洲合同纠纷案件参考

> 本文件供 legal-researcher 和相关 agent 在处理欧洲法域合同纠纷案件时按需加载。

## 核心法律框架

| 法律渊源 | 适用场景 |
|---------|---------|
| CISG（国际货物买卖合同公约） | 跨境货物买卖（中欧/美欧等均适用） |
| 德国：BGB（民法典） | 德国法管辖的合同 |
| 法国：Code civil（2016年债法改正） | 法国法管辖的合同 |
| 荷兰：BW（Burgerlijk Wetboek） | 荷兰法管辖的合同 |
| 罗马 I 法规（Regulation (EC) 593/2008） | 合同准据法选择规则（EU 统一冲突法） |
| 布鲁塞尔 I bis 法规 (Regulation (EU) 1215/2012) | 民商事管辖权与判决承认执行 |
| 不公平条款指令 (93/13/EEC) | 消费者合同中的不公平条款规制 |
| 电子识别规则 (eIDAS) | 电子签名、电子合同效力 |

## 合同效力

| 效力类型 | 认定标准（大陆法系共性） |
|---------|------------------------|
| 无效 | 违反禁止性法律（contra bonos mores / gesetzwidrig）/ 违反公共秩序（ordre public） |
| 可撤销 | 错误（erreur / Irrtum）、欺诈（dol / Täuschung）、胁迫（violence / Drohung） |
| 形式无效 | 违反法定形式要求（如不动产买卖的公证形式）|
| 显失公平（lésion / Wucher） | 利用对方困境获取显著不相当的给付 |

## 常见争议焦点

- 合同解除：法定解除（德国 BGB §323 — 催告后无果）vs 司法解除（法国 — 需法院介入，但 2016 年改正后转为单方通知解除 + 比例性审查）
- 损害赔偿范围：实际损失 + 可得利益（但以可预见性为限 — 法国 Code civil art. 1231-3；Hadley v Baxendale 规则在普通法系成员国）
- 违约金：法院可调整（德国 BGB §343 — 过高违约金可减至合理额 / 法国 — 明显过高或过低可调整，art.1231-5）
- 情势变更（Hardship / imprévision）：德国（Wegfall der Geschäftsgrundlage）→ 合同调整或解除 / 法国（2016 年 art. 1195 引入）
- Bonus Pater Familias 原则：合同解释参照合理第三人
- 竞合（contract-tort concurrence）— 法国承认非竞合原则（non-cumul），德国和荷兰承认竞合

## 合同审查重点（供 evidence-mapper 合同审查模式使用）

- 无效率由：是否违反公共秩序或禁止法律
- 歧义风险：contra proferentem 规则（对施加方不利解释 — 尤消费者合同）
- 举证风险：主张者举证原则
- 管辖风险：布鲁塞尔 I bis 规则下的审判籍
- 付款风险：同时履行抗辩（exception d'inexécution / Einrede des nichterfüllten Vertrags）
- 违约风险：实际履行（specific performance）在 civil law 国家为原则（非例外 — 与普通法相反）
- 履约风险：履约期限的确定性和合理期限
- 撤销风险：错误 / 欺诈 / 显失公平的认定标准

## 诉讼时效（统一 → 各国有差异）

| 国家 | 合同时效 |
|------|---------|
| 德国 | 3 年（BGB §195 — 年末 deadline） |
| 法国 | 5 年（Code civil art. 2224） |
| 荷兰 | 5 年（BW 3:307） |
| CISG | 公约无规定 → 各缔约国适用本国时效 |
