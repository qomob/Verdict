# 欧洲知识产权案件参考

> 本文件供 legal-researcher 和相关 agent 在处理欧洲知识产权案件时按需加载。

## 核心法律框架

| 法律/法规 | 适用场景 |
|---------|---------|
| EU Trade Mark Regulation (EUTMR 2017/1001) | 欧盟商标（EUTM — 统一效力覆盖所有成员国） |
| Trade Mark Directive (2015/2436) | 各成员国商标法统一标准 |
| Community Design Regulation (6/2002) | 欧盟外观设计（Registered and Unregistered） |
| Unitary Patent (EU 1257/2012) + UPC Agreement | 统一专利法院 — 单一效力欧洲专利 |
| European Patent Convention (EPC) | 欧洲专利（非 EU 条约 — 38 缔约国） |
| Copyright Directive (2001/29/EC — InfoSoc) | 著作权基本标准 |
| EU Copyright in the Digital Single Market (DSM Directive 2019/790) | 数字平台著作权 |
| Trade Secrets Directive (2016/943) | 商业秘密保护统一标准 |
| Enforcement Directive (2004/48/EC) | IP 执法的统一措施 |
| GDPR | 隐私与数据保护 — 与 IP 执行交叉 |

## 常见争议焦点

### 商标权
- EUIPO（欧盟知财局 — アリカンテ所在）注册 + EUTM 统一效力
- 商标类似判断（global assessment — 音形义 + 商品/服务类似性）
- 不使用取消：5 年未在 EU 真实使用
- 著名商标的跨类保护

### 专利权
- **统一专利法院（UPC — 2023年6月运营开始）** — 可选的统一管辖权
  - 一审：地方分庭（local division）+ 中央分庭（Paris / Munich / Milan）
  - 二审：卢森堡上诉法院
- EPO（欧洲专利局 — ミュンヘン）审查和异议程序
- 各国法院保留 transitional period（7+7 年）的并存管辖权

### 著作权
- 原创性标准（CJEU Infopaq C-5/08 — "author's own intellectual creation" = EU 统一标准）
- InfoSoc 指令的例外与限制
- DSM 指令 (2019) 的平台责任（Article 17 — content filtering obligation）
- 「追及权（droit de suite）」 — 艺术品的转售版税

### 商业秘密
- Trade Secrets Directive（2016/943）— 3 要件：秘密性 + 商业价值 + 合理保密措施
- 非法获取/使用/披露 + 明知的救济

## 举证要点

| 争议类型 | 关键证据 |
|---------|---------|
| EUTM 侵权 | EUTM 注册证明、使用证据、混淆调查（survey） |
| 统一专利 | EP 授权书 + UPC opt-out / opt-in 状态 |
| 著作权 | 创作记录、发表记录 |
| 商业秘密 | 保密措施的证明、接触记录 |

## 诉讼时效

| 类型 | 一般原则 |
|------|---------|
| IP 损害赔偿 | 各国法适用（通常 3-6 年） |
| 不当利得返还 | 各国法 |

## 程序特点

- EU 商标/设计的 invalidity action → EUIPO 行政程序 + 上诉至 EU General Court / CJEU
- UPC 统一专利法院 — 集中管辖，可选择德语/法语/英语为程序语言
- CJEU（Court of Justice of the European Union）负责 EU 法律的统一解释
- 跨境禁令的历史展望：CJEU Nokia v Wardell (C-316/05) 后禁令限于各国境内
