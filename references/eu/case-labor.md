# 欧洲劳动案件参考

> 本文件供 legal-researcher 和相关 agent 在处理欧洲劳动纠纷案件时按需加载。

## 核心法律框架

| 法律/指令 | 适用场景 |
|---------|---------|
| 工作时间指令 (2003/88/EC) | 每周 max 48 小时、休息、带薪年假（4 周） |
| 大量解雇指令 (98/59/EC) | 集体裁员时雇主的信息与协商义务 |
| 企业转让指令 (2001/23/EC — TUPE/Acquired Rights) | 企业转让时劳动合同自动转移 |
| 框架平等指令 (2000/78/EC) | 禁止宗教、年龄、残疾、性取向歧视 |
| 男女平等指令 (2006/54/EC — Recast) | 同工同酬、职业社会保障 |
| 临时派遣指令 (2008/104/EC) | 派遣劳动的平等对待 |
| 工作条件透明指令 (2019/1152) | 雇主以书面形式提供劳动条件信息 |
| 德国：KSchG（解雇保护法）、BGB §622 等 | 解雇保护（德国是全国最强的解雇保护法之一） |
| 法国：Code du travail | 劳动法典 |
| 荷兰：BW Book 7 Title 10 | 劳动合同 |
| GDPR | 员工数据保护 |

## 解雇保护（主要国家对比）

### 德国（最强）
- **KSchG（解雇保护法）** 适用：企业雇员 > 10 人 + 工作 6 个月以上
- 解雇需有"社会正当性"（sozial gerechtfertigt — betriebsbedingt / personenbedingt / verhaltensbedingt）
- 整理解雇（betriebsbedingte Kündigung）：必须进行 Sozialauswahl（社会选择 — 比较年龄、工作年限、扶养义务、残疾）
- 不当解雇 → 劳动关系存续 + 中间工资（Annahmeverzugslohn）
- 通常诉讼结果以补偿金（Abfindung — 月薪 × 0.5 × 工作年限）调和结案

### 法国
- 经济性解雇（licenciement économique）: 严格的实体审查 + 集体程序（PSE — plan de sauvegarde de l'emploi）
- 个人解雇（licenciement pour motif personnel）: 真实且严重的原因（cause réelle et sérieuse）
- 无正当理由解雇 → 劳动法院裁定补偿金（indemnité pour licenciement sans cause — barème Macron 2017 年改定上限和下限）
- 解雇无效（nullité）：基本権侵害（歧视、骚扰被害者保护等）→ 复职或 higher damages

### 荷兰
- 解雇需 UWV（行政机构）事前许可（经济原因），或经 kantonrechter 裁定（个人原因）
- 过渡补偿金（transitievergoeding — 1/3 月薪/年）
- 反復不当解雇：billijke vergoeding（公平补偿 — 可高于法定表）

## 举证责任

| 争议类型 | 举证方 |
|---------|--------|
| 解雇理由（德国） | 雇主证明社会正当性 |
| 歧视（EU） | 被歧视方提出 prima facie evidence → 雇主证明非歧视原因 |
| 工资支付 | 雇员证明未收到 → 雇主有工资单义务 |
| 超时劳动 | 雇员主张 → 雇主举证（工作时间记录义务 — CJEU C-55/18 CCOO v Deutsche Bank 2019） |

## 常见争议焦点

- 解雇的正当性（最大争议量）
- 经济性解雇的重雇义务
- 企业转让时劳动关系的维持
- 男女同工同酬
- 年龄歧视与退休年龄
- 平台劳动者的雇佣地位（employee vs self-employed — 各国法院和 ECJ 活跃判断）
- Whistleblower 保护（Directive 2019/1937）

## 程序特点

- **德国劳动法院（Arbeitsgericht）** 为独立序列（三审：ArbG → LAG → BAG）
- **法国 Conseil de prud'hommes**（劳资对等选出的非职业法官）
- GDPR 对雇主监控雇员的限制和员工证据收集的影响
