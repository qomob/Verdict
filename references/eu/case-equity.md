# 欧洲公司/股权纠纷案件参考

> 本文件供 legal-researcher 和相关 agent 在处理欧洲公司/股权纠纷案件时按需加载。

## 核心法律框架

| 法律渊源 | 适用场景 |
|---------|---------|
| EU Company Law Directives（多个指令） | 各成员国公司法的协调标准 |
| 德国：AktG（股份法）+ GmbHG | 股份有限公司和有限责任公司 |
| 法国：Code de commerce | 商事公司法 |
| 荷兰：BW Book 2 | 法人/公司法（灵活度高 — 许多跨国公司选用荷兰 BV/NV） |
| Societas Europaea (SE) Regulation (2157/2001) | 欧洲公司 |
| 市场濫用法规 (MAR — Regulation 596/2014) | 内幕交易、市场操纵 |
| 收购指令 (2004/25/EC) | 上市公司收购的平等对待和信息公开 |

## 公司治理体系

### 单层制 vs 双层制
- **单层制（One-tier）：** 英国（Board of Directors — executive + non-executive），法国可选（PDG + conseil d'administration）
- **双层制（Two-tier）：** 德国（Vorstand 执行董事会 + Aufsichtsrat 监事会）+ 大企业中员工共同决定（Mitbestimmung）
- 荷兰、法国允许公司自行选择

### 董事义务
- **德国：** §93 AktG — 注意义务（Sorgfaltspflicht）+ 忠实义务（Treuepflicht）
- **法国：** devoir de diligence, devoir de loyauté
- **Business Judgment Rule 的欧洲版变种：** 德国（AktG §93 — legal safe harbor if good faith + proper information + corporate benefit）

## 常见争议焦点

- 股东决议的无效（Nichtigkeit）与可撤销（Anfechtung — 德国：AktG §241 ff. 1 个月时效）
- 增资 / Das Bezugsrecht（认购权）的排除（Bezugsrechtsausschluss）
- 股东代表诉讼（Aktionärsklage — 德国：少数股东权 / 2011年法国改正引入 action ut singuli）
- 公司合并（Fusion / Verschmelzung）+ 少数股东的定价审查（Squeeze-out / 挤出）
- 资本维持原则（Kapitalerhaltung）
- 关联交易与利益冲突

## 举证要点

| 争议类型 | 关键证据 |
|---------|---------|
| 股东决议无效/可撤销 | 株主総会召集通知、决议录、投票记录 |
| 代表诉讼 | 先诉请求书、特别审查报告 |
| 挤出/价格公正性 | 企业评价报告（Unternehmensbewertung — IDW S1 或类似标准） |

## 特别注意

- **德国公司治理守則（DCGK — comply or explain）**
- **荷兰「结构体制」（structuurregime）** — 大企业的强制监事会制度
- **EU 层面的跨境合并和跨境转变指令**（Cross-border Mobility Directive 2019/2121）
- 企业评价在国际仲裁（Investment Arbitration / ICSID）中与国内 litigation 的交叉
