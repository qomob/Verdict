# 欧洲侵权责任案件参考

> 本文件供 legal-researcher 和相关 agent 在处理欧洲侵权案件时按需加载。

## 核心法律框架

| 法律渊源 | 适用场景 |
|---------|---------|
| 产品責任指令 (85/374/EEC, 1999/34/EC) | 缺陷产品所致损害的严格责任 — EU 统一标准 |
| 罗马 II 法规 (Regulation (EC) 864/2007) | 非合同债务的准据法（侵权冲突法） |
| 德国 BGB §823〜853 等 | 德国侵权法（Verschuldenshaftung / Gefährdungshaftung） |
| 法国 Code civil art. 1240〜1244（旧 1382-1386） | 法国侵权法（générique delict 条款） |
| 荷兰 BW Book 6 | 荷兰侵权法（onrechtmatige daad） |
| GDPR（Regulation 2016/679, art. 82） | 违反 GDPR 造成损害的非物质损害赔偿 |
| AI 責任指令草案 (COM/2022/496) | AI 产品責任（立法进行中） |

## 侵权类型与构成

### 德国体系
- **BGB §823 I：** 故意/过失侵害受保护法益（生命/身体/健康/自由/财产/其他权利）
- **BGB §823 II：** 违反保护性法规（如刑法条文的保护法）
- **BGB §826：** 违反善良风俗的故意侵害
- 因果关系：责任成立因果（haftungsbegründende Kausalität）+ 责任范围因果（haftungsausfüllende Kausalität）
- 损害赔偿范围：完全赔偿原则（Totalreparation）
- 精神损害（Schmerzensgeld）：仅在有法律明文时（§823 I 、§826 等）

### 法国体系
- **Code civil art. 1240：** 极开放的一般条款 —「任何人的过错造成他人损害者，应赔偿」
- 三个要素：faute（过错）+ dommage（损害）+ lien de causalité（因果关系）
- 无独立"不法性"审查，过错本身就是不法
- 精神损害赔偿范围广泛

### 荷兰体系
- **BW art. 6:162：** 侵权三要件 — 不法性 + 归责事由（过错或法律归责）+ 损害 + 因果关系
- Kelderluik criteria（cellar hatch factors）= 注意义务的判断基准

## 损害赔偿

| 类型 | 内容 |
|------|------|
| 实际损害 | 财产损害（damnum emergens）+ 可得利益丧失（lucrum cessans） |
| 非物质损害 | 德国（Schmerzensgeld — 痛苦金，限于法律条文记载的情形）、法国/比利时（广泛适用） |
| 惩罚性赔偿 | 大陆法系原则上不承认 |
| 过失相抵 | 受害人共同过失 → 比例减少赔偿 |

## 产品責任指令 (85/374/EEC) — 严格責任

- 制造者对缺陷产品所致损害承担无过错责任
- 消费者证明：损害 + 产品缺陷 + 因果关系的存在
- 制造者抗弁：開発危険の抗弁（development risk defence — Member State 可选排除）
- 赔偿上限：各成员国可以选择设定（德国、法国未设上限）
- 时效：损害日起 3 年 / 产品交付日起 10 年

## 证明标准

- 德国：Überzeugung（法官确信 — 不能排除所有怀疑但须 practical certainty）
- 举证责任缓和：Anscheinsbeweis（表见证明）/ Beweislastumkehr（举证责任转换 — 如产品缺陷推定）
- 法国：自由心证 + 过错推定の运用（如物之监管者责任）
- 荷兰：redelijke mate van zekerheid（合理程度的确定性）
