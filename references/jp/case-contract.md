# 日本合同纠纷案件参考

> 本文件供 legal-researcher 和相关 agent 在处理日本合同纠纷案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 民法（債権編） | 合同通则、买卖、赠与、租赁、雇佣、承揽等 |
| 商法 | 商事合同（商人间买卖特则等） |
| 借地借家法 | 不动产租赁 |
| 消費者契約法 | 消费者合同（信息不对称保护、不当条款规制） |
| 電子消費者契約法 | 电子合同 |

## 合同效力

| 效力类型 | 认定标准 |
|---------|---------|
| 无效 | 违反强制性法规、违反公序良俗（公序良俗違反—民法90条） |
| 可撤销 | 错误（要素の錯誤）—民法95条、欺诈/胁迫（詐欺・強迫）—民法96条 |
| 无权代理 | 本人追认前效力待定（民法113条） |

## 常见争议焦点

- 债务不履行（違約）— 履行请求 + 损害赔偿（民法415条）
- 合同解除：催告解除 + 无催告解除（不能履行/拒绝履行/合同目的不能达成）
- 瑕疵担保 vs 契約不適合責任（2020年民法改正：瑕疵担保 → 契約不適合責任）
- 违约金（違約金）— 推定损害赔偿额，不得过高（如过高可依公序良俗减少）
- 定金（手付）：解约定金（解約手付 — 放弃或加倍返还即可解除）vs 违约定金（違約手付）
- 保证（保証）：个人保证需公证（2020年改正）
- 合同解释 • 契约自由原则 + 诚实信用原则（信義則 — 民法1条2項）

## 合同审查重点（供 evidence-mapper 合同审查模式使用）

- 无效率由：公序良俗違反（如过度限制自由、暴利行为）、强行法规违反
- 歧义风险：条款模糊，对经营者不利解释（約款規制適用時）
- 举证风险：权利发生事实由主张方举证
- 管辖风险：专属合意管辖的限制（消费者合同中可能无效）
- 付款风险：同时履行抗辩权（同時履行の抗弁権 — 民法533条）
- 违约风险：损害赔偿范围（通常損害 + 特別損害但以预见可能为限 — 民法416条）
- 履约风险：不確実な場合の履行拒絶権
- 撤销风险：错误无重大过失时可主张（民法95条改正后）。消费者合同中的不实告知、断定的判断提供

## 诉讼时效（2020年民法改正后）

| 类型 | 时效 |
|------|------|
| 一般债权 | 5 年（知道权利可行使）/ 10 年（权利可行使） |
| 商事买卖价款 | 施行前 2 年 / 施行后适用 5 年 |
| 人身损害 | 5 年（知道损害和加害者）/ 20 年（行为时） |
