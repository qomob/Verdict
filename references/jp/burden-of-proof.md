# 日本举证责任与证据规则

> 本文件供 judge 和 case-intake 在评估日本法管辖案件时按需加载。

## 通用原则

### 举证责任（証明責任）

日本民事诉讼法采用 **法律要件分类说**（法律要件分類説）：
- 主张权利发生的一方 → 举证权利发生的要件事实
- 主张权利妨碍/消灭的一方 → 举证妨碍/消灭的要件事实

这是理论上的通说和判例立场（最高裁昭和33年等确认）。

### 证明标准

| 标准 | 适用场景 | 说明 |
|------|---------|------|
| 高度の蓋然性（高度盖然性） | 一般民事案件 | 法官形成确信，证明度 ≈ 70-80% |
| 疎明 | 保全处分、程序性事项 | 一応の証明（基本可信即可），低于高度盖然性证明 |

## 举证责任转换

| 特殊类型 | 举证规则 |
|---------|---------|
| 医疗过失 | 如病历等证据在医疗方，可减轻患方举证责任（最高裁平成4年 —「一応の推定」理论） |
| 产品责任（製造物責任法） | 原告证明缺陷 + 损害 + 因果关系 → 被告证明「開発危険の抗弁」等免责事由 |
| 公害/环境 | 流行病学因果关系代替个别因果关系证明（四大公害诉讼） |
| 不法行为过失 | 过失的客观化、注意义务的存在由原告举证，但可根据事实推定（事実上の推定） |

## 证据法则

### 证据能力
- 违法收集证据（違法収集証拠）— 判例区分：民事诉讼中原则上不当然排除，但如果取证方式严重违法（如侵犯人格权），个别排除
- 传闻证据 — 日本民事诉讼法无严格传闻排除规则，书证（書証）广泛采用，证言（証言）以直接主义为原则

### 证据力（証明力）
- 自由心証主義（自由心证主义，民訴法247条）— 法官自由判断证据价值
- 文书：处分文书（処分証書 — 如合同）通常证明力较高 / 报告文书（報告証書）自由评价
- 当事人陈述：补充性证据（民訴法207条 — 仅在无其他证据时可申请当事人询问）

## 证据收集手段

| 手段 | 说明 |
|------|------|
| 文書提出命令（民訴法220条） | 法院命令对方或第三人提交特定文书（范围广，包括义务文书、法律关系文书等） |
| 当事者照会（民訴法163条） | 一方可直接向对方书面询问（无强制力，不回答无 penalty，但可能产生不利心证） |
| 証拠保全（民訴法234条） | 起诉前预先固定证据 |
| 弁護士会照会（弁護士法23条の2） | 律师通过律师协会向公共机构调查 |

## 诉讼时效（消滅時効）

| 类型 | 时效 | 依据 |
|------|------|------|
| 一般债权 | 5 年（知道权利可行使时）/ 10 年（权利可行使起） | 民法166条（改正后） |
| 人身损害赔偿 | 5 年（知道损害和加害者）/ 20 年（行为时） | 民法724条（改正后） |
| 工资请求 | 3 年 | 労基法115条 |
| 商事债权 | 5 年 | 商法522条 |
| 不法行为（一般） | 3 年（知道损害和加害者）/ 20 年（行为时） | 改正前民法724条（过渡期适用） |

## 程序特点

- 三审制：地裁 → 高裁 → 最高裁
- 上告审（最高裁）为法律审，原则上只审查宪法违反和判例违反
- 労働審判制度 — 劳动争议快速解决程序（3 期内结审）
- 诉额 140 万円以下由简易裁判所管辖
