# 日本侵权责任案件参考

> 本文件供 legal-researcher 和相关 agent 在处理日本侵权案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 民法709条〜724条 | 不法行为（侵权）基本规则 |
| 製造物責任法（PL法） | 产品责任 — 严格责任 |
| 自動車損害賠償保障法（自賠法） | 交通事故 — 运行供用者责任（近乎无过错） |
| 国家賠償法 | 国家/公共团体赔偿责任 |

## 不法行为成立要件（民法709条）

1. **故意または過失（故意或过失）** — 过失判断客观化，违反注意义务
2. **権利または法律上保護される利益の侵害** — 权利或受法律保护的利益被侵害
3. **損害の発生** — 损害发生（财产损害 + 精神损害（慰謝料））
4. **因果関係** — 事实因果关系（conditio sine qua non）+ 相当因果关系（赔偿范围的限定）

## 特殊不法行为

| 类型 | 内容 | 条文 |
|------|------|------|
| 使用者责任 | 雇主对雇员执行职务所致损害负责（免責抗弁极难成立） | 民法715条 |
| 工作物责任 | 土地工作物瑕疵所致损害，占有人/所有人责任 | 民法717条 |
| 共同不法行为 | 共同侵权人对全部损害承担不真正连带责任 | 民法719条 |
| 监督者责任 | 无责任能力人（未成年人）所致损害，监督义务人负责 | 民法714条 |
| 产品责任 | 製造物責任法（PL法）— 制造者等严格责任。抗弁：開発危険の抗弁 | PL法 |

## 损害赔偿

| 损害赔偿类型 | 内容 |
|-------------|------|
| 積極損害（积极损害） | 医疗费、护理费、交通费等已支出费用 |
| 消極損害（消极损害） | 逸失利益（丧失的未来收入 — 可計算） |
| 慰謝料（精神损害） | 入通院慰謝料 + 後遺障害慰謝料 + 死亡慰謝料（参照「民事交通事故訴訟・損害賠償額算定基準」— 赤本/青本基准） |
| 過失相殺（与有过失） | 受害方过失 → 减少赔偿额（民法722条2項） |
| 損益相殺 | 抵扣已获利益（如有） |

## 诉讼时效

| 类型 | 时效 |
|------|------|
| 不法行为（2020年民法改正后） | 5 年（知道损害和加害者）/ 20 年（行为时） |
| 人身损害 | 同上（不再区别对待） |
| 产品责任 | 3 年（知道损害和加害者）/ 10 年（交付时起） |

## 证明标准

- 一般民事侵权：高度の蓋然性（80% 程度の心証）
- 因果关系的证明：不要求自然科学的精确证明，但需达到经验法则上的高度盖然性
- 医疗诉讼中的证明减轻：可通过「一応の推定」理论（高度盖然性进一步降低）

## 主要判例

| 判例 | 要点 |
|------|------|
