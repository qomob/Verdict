# 日本知识产权案件参考

> 本文件供 legal-researcher 和相关 agent 在处理日本知识产权案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 著作権法 | 著作权 |
| 特許法 | 发明、实用新型（実用新案法） |
| 商標法 | 商标 |
| 意匠法 | 外观设计 |
| 不正競争防止法 | 商业秘密、商品表示混同、形态模仿、域名不当取得 |

## 常见争议焦点

### 著作权
- 著作物性（独创性 — 最低限度即可）
- 著作者人格権（公表权、氏名表示权、同一性保持权）
- 侵害认定：依拠性（接触）+ 類似性
- 私的使用复制例外（30条）
- 电影著作物的著作権帰属

### 商标权
- 商标类似判断：外観・称呼・観念三者综合判断
- 商标使用：需在指定商品/服务上使用
- 不使用取消审判（3年以上未在日本使用 — 不使用取消審判）
- 著名商标的防護標章制度
- 商标権侵害 → 民事（差止請求 + 損害賠償）+ 刑事

### 专利权
- 進歩性（inventive step）判断
- 均等論（doctrine of equivalents — 最高裁平成10年 Ball Spline 事件 5 要件）
- 特許無効審判 vs 侵害诉讼中的无效抗辩（キルビー最高裁判決 + 2004年法改正引入）
- 损害赔偿额计算：特許法102条 — 侵害者利益推定为损害额 / 実施料相当額 / 超过部分

### 商业秘密（営業秘密）
- 不正競争防止法上的 3 要件：秘密管理性 + 有用性 + 非公知性
- 民事：差止請求 + 損害賠償 + 信用回復措置
- 刑事：営業秘密侵害罪（2015年改正强化）

## 举证要点

| 争议类型 | 关键证据 |
|---------|---------|
| 著作权侵权 | 創作の事実（创作底稿）、依拠証拠、类似性证据 |
| 商标侵权 | 商标注册证、使用证据、知名度证据 |
| 专利侵权 | 特許公報、クレームチャート、被控侵权产品、无效资料 |
| 商业秘密 | 秘密管理規程、アクセスログ、NDA |

## 诉讼时效

| 类型 | 时效 |
|------|------|
| 知识产权不法行为损害赔偿 | 3 年（知道损害和加害者）/ 20 年（行为时） |
| 不当利得返还 | 5 年 / 10 年 |

## 程序特点

- 知的財産高等裁判所（知识产权高等法院 — 东京高裁特别支部，专门管辖）
- 专利侵权诉讼集中管辖：东京地裁 + 大阪地裁（双方 29部/26部 为专业部）
- 裁判所调查官制度（技术背景调查官辅助法官理解技术问题）
- 侵害論と損害論分離审理（侵权认定和损害赔偿计算分开）
