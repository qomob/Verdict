# 日本劳动案件参考

> 本文件供 legal-researcher 和相关 agent 在处理日本劳动纠纷案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 労働契約法 | 劳动合同基本规则 |
| 労働基準法 | 最低劳动条件（工资、工时、解雇） |
| 労働組合法 | 工会权利、不当劳动行为 |
| 雇用機会均等法 | 男女就业机会平等、性骚扰 |
| 育児・介護休業法 | 育儿/照护休假 |
| 労働安全衛生法 | 工作场所安全 |
| 最低賃金法 | 最低工资 |
| パートタイム・有期雇用労働法 | 非正规雇佣（兼职、固定期限）同工同酬 |

## 解雇法理（極めて重要）

日本采用**解雇権濫用法理**（労働契約法第16条）：
- 解雇需有「客観的に合理的な理由」和「社会通念上相当」两个条件
- 普通解雇（ordinary dismissal）：需客观合理理由
- 整理解雇（economic/restructuring dismissal）四要件：人员削减的必要性 + 努力避免解雇 + 被解雇者选择的合理性 + 程序适当性
- 懲戒解雇：需要更重的理由
- 解雇无效 → 劳动关系存续 + 中间工资全额支付（労働基準法第26条）+ 复职（原职恢复）

## 举证责任

| 争议类型 | 举证方 |
|---------|--------|
| 解雇理由 | 雇主（解雇权的存在 + 滥用的不存在均由雇主举证） |
| 未支付工资 | 劳动者证明存在劳动 + 工资约定 → 雇主承担支付事实的举证 |
| 加班费 | 劳动者主张 → 雇主有出勤记录保存义务 |
| 性骚扰 | 劳动者证明骚扰事实 → 雇主证明已尽防止义务 |
| 工伤（労災） | 劳动者向劳动基准监督署申请认定（行政程序） |

## 常见争议焦点

- 解雇无效确认 + 地位确认请求
- 雇止め（固定期限合同不续签）— 如有「期待権」，拒绝更新需合理理由
- 未払い残業代（unpaid overtime）请求 — 追溯 2-3 年（附则暂时 3 年）
- 職種や勤務地限定正社員（limited regular employee）
- パワハラ（power harassment / workplace bullying）
- 同一労働同一賃金（equal pay for equal work — 2020年施行）

## 救济方式

| 救济 | 说明 |
|------|------|
| 地位確認 | 确认劳动关系存续 |
| 賃金仮払い仮処分 | 临时工资支付保全处分 |
| 未払賃金＋付加金 | 未支付工资 + 等额附加金（労基法114条） |
| 慰謝料 | 精神损害赔偿 |

## 程序特点

- 劳动审判制度（労働審判 — 2006年设立，原则 3 回期日以内结审）— 介于调解和诉讼之间的快速程序
- 劳动局/劳动基准监督署的行政救济（斡旋、是正指导）
- 劳动组合（工会）通过団体交渉（group negotiation）解决争议
- 地方法院劳动专业部集中管辖

## 诉讼时效

| 类型 | 时效 |
|------|------|
| 工资请求 | 3 年（退职金 5 年）— 労基法115条 |
| 解雇无效 | 除斥期间无，但合理期限内行使 |
| 损害赔偿（不法行为） | 3 年（知道损害和加害者）/ 20 年（行为时起）— 民法724条 |
