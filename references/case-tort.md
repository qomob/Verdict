# 侵权责任案件参考

> 本文件供 legal-researcher 和相关 agent 在处理侵权责任案件时按需加载。

## 核心法律框架

| 法律 | 适用场景 |
|------|---------|
| 《民法典》侵权责任编（第七编） | 一般侵权、特殊侵权、损害赔偿 |
| 《消费者权益保护法》 | 产品责任、服务侵权 |
| 《道路交通安全法》 | 交通事故责任 |
| 《环境保护法》 | 环境侵权责任 |

## 归责原则

| 原则 | 适用场景 | 举证责任 |
|------|---------|---------|
| 过错责任 | 一般侵权 | 受害人证明侵权人有过错 |
| 过错推定责任 | 建筑物倒塌、医疗损害等 | 侵权人证明自己无过错 |
| 无过错责任 | 高度危险作业、环境污染、产品责任 | 不论侵权人是否有过错 |

## 举证责任倒置

| 案件类型 | 倒置内容 |
|---------|---------|
| 医疗损害 | 医疗机构证明无过错、无因果关系 |
| 环境污染 | 污染者证明行为与损害无因果关系 |
| 高度危险作业 | 经营者证明受害人故意 |
| 饲养动物损害 | 饲养人证明受害人故意或重大过失 |

## 常见争议焦点

- 侵权行为与损害结果之间的因果关系
- 损害赔偿范围（医疗费、误工费、护理费、残疾赔偿金、精神损害抚慰金）
- 多人侵权的责任分担（连带 vs 按份）
- 精神损害赔偿的认定标准
- 网络侵权责任（通知删除规则）
- 共同侵权（主观意思联络 → 连带责任）
- 分别侵权（各自行为竞合 → 按份或连带）

## 证明标准

一般民事侵权适用"高度盖然性"标准（>75%）。医疗损害等专业技术性强的案件常依赖鉴定意见。
