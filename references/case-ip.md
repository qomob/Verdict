## 常见争议焦点

### 著作权
- 权属认定（原始取得 vs 继受取得）
- 侵权比对（实质性相似 + 接触）
- 合理使用认定（四要素法）

### 商标权
- 商标相同/近似认定（音形义综合比对）
- 商品/服务类似认定
- 混淆可能性判断
- 驰名商标（跨类保护、淡化理论）

### 专利权
- 保护范围界定（权利要求书解释）
- 全面覆盖原则
- 等同侵权（以基本相同的手段实现基本相同的功能）

### 商业秘密
- 秘密性（不为公众所知悉）
- 价值性（具有商业价值）
- 保密措施（采取了合理保密措施）

## 举证要点

| 争议类型 | 关键证据 |
|---------|---------|
| 著作权侵权 | 权属证明（登记证书/创作底稿）、侵权证据（公证书/时间戳） |
| 商标侵权 | 商标注册证、侵权商品/行为证据、知名度证据 |
| 专利侵权 | 专利证书、权利要求书、被控侵权产品技术特征比对 |
| 商业秘密 | 保密协议、访问日志、信息载体、侵权人接触证据 |

## 赔偿计算

赔偿额计算顺序：权利人实际损失 → 侵权人获利 → 许可费倍数 → 法定赔偿（著作权/商标 500元-500万，专利 3万-500万）+ 合理开支（律师费、公证费）。
